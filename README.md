# Inline-4-Internal-Combustion-Engine-Solidworks
A fully modeled inline 4-cylinder (I4) engine designed in SolidWorks. This repository contains the complete 3D assembly, a complete motion study demonstrating engine functionability, and an engineering drawing for the isometric view along with all the components involved.

https://www.youtube.com/watch?v=se4CElm2t10

## 🔧 Software Requirements

- **SolidWorks** (2020 or later recommended)
- All part files must be present in the same directory as the assembly for it to resolve correctly

---

## 🏎️ About the Design

This project is a fully assembled inline 4-cylinder engine model built in SolidWorks. An inline 4 (I4) is one of the most common internal combustion engine configurations, featuring four cylinders arranged in a single row.

**Key engine components modeled:**

By Safwan Ikbal

- Cylinder head
- Pistons
- Connecting rod
- Connecting Rod Clamp
- Crankshaft (Single & Double Piston) Assembly
- CounterWeights
- Ring
- Full Engine Assembly

By Arav Patel:

- Engine block
- Block Clamp
- Engine Block Fly Wheel
- Engine Block Side Plates

Safwan & Arav have both contributed to reviewing and revising all parts made*

## 📐 Key Files

| File | Description |
|------|-------------|
| `FullEngineAssemby.SLDASM` | Final SolidWorks assembly containing all parts and subassemblies |
| `FullEngineAssemby_Drawing.SLDDRW` | 2D engineering drawing with isometric view and components |
| `SinglePistonAssembly.SLDDRW` | Key subassembly, used in the main 'FullEngineAssembly.SLDASM'. Involves a single piston model |
| `DoublePistonAssembly.SLDDRW` | Key subassembly, involving a double piston model |
| `ENGINEMOTIONSTUDY.mp4` | Solidworks motion study. Demonstrates engine functionability |


---

## 🚀 Getting Started

1. Clone or download this repository
2. Open `FullEngineAssemby.SLDASM` in SolidWorks
3. Ensure all referenced part files are in the correct directory
4. Use `FullEngineAssemby_Drawing.SLDDRW` to view the engineering drawing
